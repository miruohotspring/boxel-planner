# 構造パターンから選ぶ

「どんな構造を置きたいか」から逆引きするときの入口です。

## 左右対称・前後対称

- 片側だけ作って `mirror`
- 部品ごと反転して置くなら `place --mirror`

## 繰り返し配置

- 窓、柱、胸壁、アーチ列は `copy --repeat`
- 同じ部品を複数箇所へ置くなら `place`

## 円塔・円形基壇

- 壁だけ: `cylinder`
- 床や天板: `circle --filled`
- 円形の主塔 + 尖塔: `cylinder` + `spire`

## 尖塔・段付き屋根

- 深い軒や瓦屋根は `roof`
- 中央破風は `gable`
- 一番使いやすいのは `spire`
- 角屋根や寄棟風なら `fill` の段積み

## ドーム・曲面

- 球面系: `sphere`
- 楕円平面: `ellipse`
- 数式ベースの特殊形状: `surface`

## 分業合成

- 部品ごとに別ファイルを作る
- `place` で最終図へ合成する
- 仕上げ前に `recenter` で原点を揃える

詳しいコマンド仕様は [../llm-guide.md](../llm-guide.md) を参照してください。
