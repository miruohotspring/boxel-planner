# 人間スケールと開口設計

建物が見た目として成立していても、人が使う前提で見ると破綻することがあります。
ここでは、駅舎のレビューで実際に問題になった点を、今後の共通規約としてまとめます。

## 固定ルール

- スケールは `1 block = 1 meter`
- 通行可能な段差は `1m` まで
- `2m` 以上の段差は通行不可能として扱う
- 出入口の開口高さは `2m` に統一する
- 内部の実行動空間の高さは最低 `3m` 確保する
- 原則は `ドア 2m / 天井 3m 以上`

`check-access` が通っていても、頭上が低ければ「人が使える建築」ではありません。

## 設計で気を付けること

### 1. グラウンドレベルを先に揃える

駅舎、ホーム、広場、通路の床レベルがバラバラだと、局所的にはつながって見えても実用性が崩れます。

- まず主要動線の床高さを決める
- 同じ用途で連続して歩く面は、極力同じ高さに揃える
- やむを得ず高低差を作る場合でも、1m 刻みで解消する

今回の駅舎では、正面広場と駅舎入口の関係、駅舎床とホーム床の関係が最初に破綻しやすかったです。

### 2. 階段は「見た目」より「接続」を優先する

大階段は見栄えがしますが、建物から離れていたり、段差が大きすぎると機能しません。

- 階段は入口に接続していることを優先する
- 1 段 = 1m を超える段差にしない
- 入口の正面導線を外さない
- 階段を作るなら、踊り場や前面テラスとの関係も見る

「駅前に大きな階段がある」こと自体は問題ではなく、「その階段を上っても入口に自然につながらない」状態が問題です。

### 3. 出入口以外に外部へ抜ける穴を作らない

内部空間を空けようとして、壁の最下段や側面まで抜けてしまうことがあります。

- 出入口として意図した場所だけを開ける
- 窓は窓として残し、通路穴と混同しない
- `--hollow` を使うときは、床・天井・最下段の扱いを必ず確認する
- 開口を広げたら、同じ高さの別面にも意図しない穴が出ていないか見る

今回の駅舎では、`y=2` の処理を調整した結果、床上空間は確保できても、壁の輪郭まで消えるケースがありました。

### 4. 入口の高さと内部天井高を分けて考える

入口が広いことと、人が快適に動けることは別です。

- ドア開口は `2m`
- 室内の可動空間は `3m` 以上
- 吹き抜けやホールはさらに高くてもよい
- 低い付属室でも、可動空間が 3m を下回らないかを見る

「ドアを 3m 開ける」のではなく、「ドアは 2m、室内天井は 3m 以上」が基本です。

## レビューで問題になったポイント

以下は今回の駅舎レビューで実際に指摘された内容です。

- 正面の大階段が `3m` あり、高すぎた
- 正面や側面から、`1m` の段差を越えないと中へ入れない構成になっていた
- 駅舎からホームへの通路が高すぎて、実質的に行き来しづらかった
- 開口高さが足りず、人が使う通路としては窮屈だった
- 床上空間を空ける修正の途中で、出入口以外の抜け穴が生まれた
- 階段と建物が離れていて、入口動線として不自然だった

見た目が良くても、このどれかに当てはまるなら、まず人間スケールを疑ってください。

## 点検方法

### `check-access` だけで終わらせない

`check-access` は空セル同士の到達性しか見ません。
頭上クリアランスや「人が立てる高さ」は判定しません。

そのため、最低でも次を併用します。

1. `boxel check-access` で経路の有無を見る
2. `boxel render --y ...` で床と壁の関係を見る
3. `boxel ortho --mode coord` で立面の抜け穴や段差を確認する

### とくに見るべき断面

- `y=1`: 実際に歩く床面
- `y=2`: ドア下端のすぐ上。意図しない抜け穴が出やすい
- `y=3`: 人が通る空間として最低限ほしい高さ帯
- `front` / `side` の `coord`: 壁面の穴や不連続な開口の確認

## 実装上の注意

- `hollow` を使うときは、どの面が残るかを必ず断面で確認する
- 内部を空けるときは「床を残して空間を抜く」のであって、「壁の輪郭まで消す」わけではない
- 低い付属室やサービス区画は、意図がない限り開放空間にしない
- 入口を広げる前に、まず高さと床レベルを揃える
- 迷ったら、床面と出入口だけを先に作ってから外観を盛る

## 今後の扱い

今後は、建築物を作るときに次を標準規約とします。

- 開口高さは `2m`
- 室内の実行動空間は最低 `3m`
- 通行可能な段差は `1m` まで
- 出入口以外の外部開口は、意図が説明できる場合だけ残す

見た目の正しさより先に、人が自然に出入りできて、中で動けることを満たしてください。
