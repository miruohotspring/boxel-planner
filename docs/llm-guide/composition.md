# 部品合成と対称配置

大きな建築は 1 ファイルで掘るより、部品化して組み立てた方が安定します。

## 基本の流れ

1. 中央塔、翼棟、門、角塔を別ファイルで作る
2. 各部品の基準点を `(0,0,0)` に揃える
3. 最終ファイルで `place`
4. 対称面は `mirror` か `place --mirror`
5. 反復要素は `copy --repeat`

## よく使うコマンド

- `place --collision error`
  - 意図しない食い込みを先に止める
- `place --rotate-y 90|180|270`
  - 門や翼棟の向きを変える
- `place --mirror x|z`
  - 左右対称や前後対称の部品配置
- `place --repeat --step-x/--step-y/--step-z`
  - 同じ部品を等間隔に並べる
- `recenter`
  - 全体を標準座標規約へ戻す

## おすすめの分割単位

- 城: 中央塔 / 翼棟 / 門 / 角塔
- 寺院: 本堂 / 回廊 / 門 / 塔
- 町並み: 建物本体 / 屋根 / 装飾帯

詳しいコマンド仕様は [../llm-guide.md](../llm-guide.md) を参照してください。
