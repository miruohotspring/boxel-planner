# プリミティブ早見表

## 直方体・壁・床

- `fill`: 直方体、床、壁、テラス、段差

## 単体ブロック

- `add`: 旗、装飾、頂点
- `remove`: 扉、窓、アーチ穴

## 円形

- `circle`: 1 層の円盤やリング
- `cylinder`: 円塔、柱、円基壇
- `spire`: 尖塔、段付き屋根、城の青屋根
- `sphere`: ドーム、球、半球
- `ellipse`: 楕円広場、楕円池

## 曲面

- `surface`: トーラス、丘、波面、鞍面など

## 配置と再利用

- `copy`: 同一ファイル内の反復
- `mirror`: 対称複製
- `place`: 別ファイル部品の配置
- `recenter`: XZ 中心 / 底面 Y=0 へ寄せる

詳しいコマンド仕様は [../llm-guide.md](../llm-guide.md) を参照してください。
